```
NAME 
 semverutil - Semantic version command line utility

SYNOPSIS 
 semverutil [ -MNP ]

DESCRIPTION 
 semverutil parses semantic version strings from STDIN and optionally
 increments the major, minor, or patch version. If no options are speci<hy>
 fied then semverutil will output the highest of the semantic version
 values given in STDIN.

 Options
 -M
   Increment the major version
 -N
   Increment the minor version
 -P
   Increment the patch version
```
